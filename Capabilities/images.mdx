## Quickstart

To generate an image from a prompt, you can use the `v1/images/generations` endpoint. Here's how you can make a request to generate an image:

<Tabs>
  <Tab title="cURL">
  ```bash
curl https://api.clashai.eu/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $CLASHAI_API_KEY" \
  -d '{
        "model": "imagen-3.0-generate-001",
        "prompt": "A cute baby sea otter",
        "n": 2,
        "size": "256x256"
    }'
  ```
  </Tab>
  <Tab title="Python">
```python
import clashaipy
import os

api_key = os.getenv("CLASHAI_API_KEY")
client = clashaipy.Client(api_key=api_key)

response = client.image.generate(
    prompt="FLUX.1 [pro]",
    n=2,
    size="1024x1024",
    model="FLUX.1 [pro]"
)

for i, image in enumerate(response.get("images", [])):
    print(f"Image {i + 1}: {image}")
```
  </Tab>
  <Tab title="JavaScript">
```javascript
const axios = require('axios');
const apiKey = process.env.CLASHAI_API_KEY;
const url = 'https://api.clashai.eu/v1/images/generations';

const data = {
  model: 'FLUX.1 [pro]',
  prompt: 'A futuristic cityscape at sunset',
  n: 2,
  size: '512x512'
};

const headers = {
  'Content-Type': 'application/json',
  'Authorization': `Bearer ${apiKey}`
};

axios.post(url, data, { headers })
  .then(response => {
    response.data.images.forEach((image, index) => {
      console.log(`Image ${index + 1}: ${image}`);
    });
  })
  .catch(error => {
    console.error('Error:', error);
  });

```
  </Tab>
</Tabs>

The response will be a JSON object with the following structure:
```json
{
	"created": 1740306117,
	"data": [
		{
			"url": "http://cdn.clashai.eu/v1/images/72820237"
		}
	]
}
```
