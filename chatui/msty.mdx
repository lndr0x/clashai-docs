## About MSTY
MSTY is an application designed to simplify the use of both local and online AI models. It offers a user-friendly installation process without requiring Docker or command-line interface knowledge.

## Prerequisites
Before starting, make sure you have:
1. An API key from ClashAI
2. The [MSTY application](https://msty.app/) installed on your device

<Steps>
  <Step title="Get API key">
  First, join our [Discord server](https://discord.gg/yyfGnyKj9j) and navigate to the [Dashboard channel](https://discord.com/channels/1231339329435996280/1302343342746505347).

  Look for the **"Manage API Key"** section in the left sidebar. Click on **"Generate"** to receive your unique API key that will enable access to our services.
  </Step>

  <Step title="Install MSTY">
  Visit the [MSTY website](https://msty.app/) and click the "Install" button to download the application.
  </Step>

  <Step title="Setting up MSTY">
  After installing MSTY, follow these steps:
  1. Click on "Add remote model provider"
  2. In the new window that opens:
     - Select "Open AI Compatible" from the "Provider" dropdown
     - Enter "ClashAI" in the "Name" field
     - Input `https://api.clashai.eu/v1` as the "API Endpoint"
     - Paste your API key in the "API Key" field
  3. Click "Fetch models" to see the complete list of available models
  4. Select the models you want to add by:
     - Checking individual checkboxes next to model names, or
     - Clicking "Select Models" to add all available models
  5. Finally, click "Add" to complete the setup
    <img src="/images/msty.png" />
  </Step>
</Steps>
